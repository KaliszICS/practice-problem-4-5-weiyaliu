# Instructions  

1. Create a function called _triangle()_ which takes in an integer as a parameter.
Return string with a triangle of asterisks(*) back to the user of that many rows.

Example:

triangle(3) returns:

		*
		**
		***

2. Create a function called _pyramid()_ which takes in an integer as a parameter.
Return string with a pyramid of asterisks(*) back to the user of that many rows.

pryamid(4) returns:

		   *
		  * *
		 * * *
		* * * *

3. Create a function called _multiplicationTable()_ which takes in an integer as a parameter.
Return a string of a multiplication table from 1 to the number.

Example:

multiplicationTable(3) returns:

		1 2 3
		2	4	6
		3	6	9
